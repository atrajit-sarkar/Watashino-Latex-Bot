Commands

/start

Meet the bot, see a demo image, and read a quick intro.

Scope: DM or server ‚Ä¢ Ephemeral responses

/latex

Render a LaTeX expression to PNG and PDF.

/latex code: "$x^2 + \int_0^1 x\,dx$"
  • Returns a crisp PNG preview and a cropped PDF for vector editing.
  • Uses your custom preamble if configured; otherwise the default.

/settings

Open an interactive panel with buttons to toggle caption code and edit your preamble via a modal.

  • Toggle ‚ÄúCode in caption‚Äù on/off with one click.
  • Open a modal to paste or edit your entire LaTeX preamble.

/setdpi

Set the DPI (image resolution) from 100 to 1000.

/setdpi dpi: 400

/getmypreamble

Show your current preamble (custom if set, or default otherwise).

/getdefaultpreamble

Display the default preamble used by the bot.

/setcustompreamble

Open a modal where you can paste your custom preamble.

Setup & Requirements

  • Bot token set in .env as DISCORD_TOKEN.
  • Ghostscript and a LaTeX distribution (pdflatex) available on PATH.
  • Optional: DISCORD_GUILD_ID for fast command sync in one server.
  • Optional: enable Message Content Intent and set DISCORD_ENABLE_MESSAGE_CONTENT=true if you need message content (not required for slash commands).
Tip: Use /settings to toggle showing raw code in the image caption and to edit your preamble via a modal.

FAQ

Why do I get a syntax error?

LaTeX compilation errors from pdflatex are shown in the bot‚Äôs response. Check for missing packages or typos. You can also try with the default preamble to narrow down issues.

How do I include my macros?

Open /settings and click Edit preamble to paste your macros and packages into a full preamble. All future renders will use it.

What‚Äôs the best DPI?

Start with 300. Use /setdpi 400 for larger displays, or 720 for very crisp previews.
